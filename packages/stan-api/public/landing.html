Account Abstraction

Tips Execute as Smart Account UserOps

STAN's tip wallet is a Safe smart account (ERC-4337). The WDK EOA owns the Safe — same BIP-44 key, promoted to smart account controller. Every tip fires as a UserOperation through Pimlico's bundler, not a raw EOA send.

01 Brain clears all 5 conviction gates — daily cap, momentum, tier, cooldown, yield guard
02 WDK withdraws USDT from Aave V3 → lands in EOA (WDK primitive)
03 WDK EOA funds the Safe smart account (tip amount transferred on-chain)
04 Safe sends USDT to creator as an ERC-4337 UserOperation via Pimlico bundler
05 UserOp hash is verifiable on-chain — full auditability, upgradeable account

WDK_SEED → m/44'/60'/0'/0/0 → EOA owner → Safe 1.4.1 (ERC-4337, EntryPoint v0.7)
Tip path: Brain → WDK Aave withdraw → EOA → Safe → UserOp → creator
